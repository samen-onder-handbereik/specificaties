# Glossary

## Uitkomst Overleg

**Status:** Werkbegrip -- kandidaat voor NL-SBB

De informatie die via de inzage-API beschikbaar wordt gesteld als resultaat van een casusoverleg.


## Betrokkene

**Status:** Werkbegrip -- kandidaat voor NL-SBB

De persoon waarop de Uitkomst Overleg betrekking heeft.


## Entity

**Status:** Technisch begrip

Een PROV-concept dat een informatieobject of ander identificeerbaar object representeert.


## Activity

**Status:** Technisch begrip

Een PROV-concept dat een proces of gebeurtenis beschrijft waardoor informatie ontstaat of wordt gebruikt.


## Agent

**Status:** Technisch begrip

Een PROV-concept dat de actor representeert die verantwoordelijk is voor een activiteit.


## Provenance

**Status:** Werkbegrip -- kandidaat voor NL-SBB

Informatie over de herkomst en het gebruik van gegevens.


## Identifier

**Status:** Technisch begrip

Een waarde waarmee een element binnen een model eenduidig kan worden onderscheiden.


## Businessidentifier

**Status:** Werkbegrip

Een identifier die betekenis heeft binnen het domeinmodel.


## Provenance identifier

**Status:** Werkbegrip

Een technische identifier waarmee een PROV-element uniek wordt aangeduid.
